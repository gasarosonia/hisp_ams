---
description: How to push changes to the GitHub repository
---

To push your changes to the project repository:

1. Ensure you are in the project root directory (`hisp_ams`).
2. // turbo
   - `git add .`
3. // turbo
   - `git commit -m "Update project contents"`
4. // turbo
   - `git push origin master`

> [!NOTE]
> If you encounter 'rejected' errors, run `git pull origin master --rebase` first.
